Technical Description

Our core technology utilizes advanced algorithms, sensors, and real-time data processing to improve urban traffic systems. Basically, we implement cameras into traffic lights. Using that cameras and computer vision algoritms, we collect data, analyse it and give real-time adjustment to the traffic lights. Our data model use different weights for different types of cars and prioritization. Below are some key technical components of our solution:

AI-Powered Traffic Light Management

Object Detection: using YOLOv8 model to detect the vehicles from cameras. Vehicle Classification: classifying detected cars into categories (cars, buses, emergency vehicles etc.) with fine-tuning the model with photos of vehicles in the particular city.

Real-Time Data Processing

Regression Models: applying XGBoost model to predict traffic timing based on vehicle counts, type of cars, traffic patterns (time of day, rush hours), and weather (applying OpenWeatherMap API for real-time weather)

Deployment

Low-Latency Processing: running models on NVIDIA Jetson which reduces the need for cloud processing and improves response time.
